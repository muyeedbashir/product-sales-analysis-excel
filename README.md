# 📊 Product Sales Analysis (Excel Project)

## 🟢 Level: Beginner Project

---

## 🔍 Objective
To analyze product sales data and understand basic data cleaning and lookup techniques in Excel.

---

## 🛠️ Tools Used
- Microsoft Excel

---

## 📌 Steps Performed

### Data Cleaning
- Removed duplicate values  
- Used TRIM function  
- Used Text to Columns  

### Data Analysis
- Used VLOOKUP  
- Used INDEX-MATCH  
- Used XLOOKUP  

---

## 🔄 Before vs After Data Cleaning

### ❌ Before Cleaning
- Duplicate records present  
- Inconsistent text formatting (extra spaces)  
- Data not properly split into columns  

### ✅ After Cleaning
- Removed all duplicate values  
- Cleaned text using TRIM function  
- Structured data using Text to Columns  
- Data ready for analysis
